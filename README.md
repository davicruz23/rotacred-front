# 🌐 RotaCred Web

Aplicação front-end desenvolvida com **React + TypeScript + Vite**, integrante do ecossistema do sistema *RotaCred*.  
Responsável pela interface web administrativa, consumindo APIs REST seguras desenvolvidas no back-end.

O projeto foi estruturado com foco em organização, separação de responsabilidades e facilidade de manutenção.

---

## 🧩 Tecnologias e Ferramentas

- **React** – Construção de interfaces modernas
- **TypeScript** – Tipagem estática e maior segurança no desenvolvimento
- **Vite** – Build tool rápida com hot-reload
- **SCSS** – Organização e modularização de estilos
- **Docker** – Containerização para ambiente isolado
- Preparado para deploy em ambientes estáticos (Netlify, Vercel ou similares)

---

## 📌 Funcionalidades Principais

✔️ Interface web responsiva  
✔️ Consumo de API REST autenticada  
✔️ Separação de componentes por responsabilidade  
✔️ Organização modular de pastas  
✔️ Configuração via variáveis de ambiente  
✔️ Estrutura preparada para crescimento e escalabilidade

## 🛠 Como Rodar Localmente

### 🔹 Pré-requisitos

- Node.js ≥ 16.x  
- npm ou yarn  

### 🔹 Passos

```bash
# Clone o repositório
git clone https://github.com/davicruz23/rotacred-web.git

# Entre na pasta
cd rotacred-web

# Instale as dependências
npm install
# ou
yarn

# Execute o projeto
npm run dev
# ou
yarn dev
